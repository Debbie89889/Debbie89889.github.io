project_01 · semantic search & ir

電商語意商品搜尋

用 Amazon ESCI 電商資料微調 bi-encoder 檢索器加 cross-encoder reranker， 把使用者的自然語言查詢對應到正確商品，即使用詞完全不同。採業界標準的兩段式 檢索（先召回再精排），核心在嚴謹的檢索品質評估。

project_02 · ml & data product

消費者智慧分析平台

project_03 · llm agent & rl

單一 SKU 補貨決策 AI Copilot

從零自建一套 LLM Agent，以 router、extractor、tool、response 四階段編排， 讓使用者用自然語言操作背後的強化學習補貨模型。主打 Agent 系統設計， 以及把 LLM 嚴格約束在敘述層，確保每個數字都來自模型計算、而非被生成出來。

4 階段
Agent 編排
數字可追溯
LLM 只敘述不計算
68→79%
服務水準提升
  • router → extractor → tool → response 的完整 Agent 編排，provider-agnostic 可切換不同 LLM
  • LLM 只翻譯不計算，每個數字都能在 trace 中追溯，輸出固定五段式決策說明
  • 以深度強化學習訓練補貨策略，表現勝過傳統 (s,S) 存貨基準
